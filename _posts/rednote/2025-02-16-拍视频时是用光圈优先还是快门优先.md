### 根据需求决定固定快门速度还是光圈

| 场景 | 固定快门速度 | 固定光圈 |
| --- | --- | --- |
| 电影感画面（自然的运动模糊） | 是（1/50s、1/60s 等） | 可调整 |
| 浅景深（背景虚化） | 可调整 | 是（f/1.8、f/2.8 等） |
| 大景深（全部清晰对焦） | 可调整 | 是（f/8、f/11 等） |
| 慢动作拍摄 | 是（如 50fps 用 1/100s，120fps 用 1/240s） | 可调整 |
| 低光拍摄 | 可调整 | 是（保持大光圈，如 f/1.8 或 f/2.8） |
| 快速运动（运动、体育） | 是（更快的快门，如 1/250s、1/500s） | 可调整 |
| 明暗变化的环境（室内外切换） | 是（快门优先模式） | 自动光圈调整 |
| 采访 / 访谈视频 | 可调整 | 是（如 f/2.8 或 f/4，以虚化背景） |
| 风景 / 广角拍摄 | 可调整 | 是（f/8-f/11，保证清晰度） |
| 摄影棚 / 受控光线环境 | 是 | 是 |
